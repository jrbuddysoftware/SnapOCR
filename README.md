# Quick Snip OCR

Free screen text extractor for Windows 10/11.

Press **Ctrl+Shift+S** anywhere, drag to select a region, and get the text copied to your clipboard instantly.

## Features

- **Instant screen capture OCR** — global hotkey works from any app
- **6-pass multi-algorithm OCR** — upscaling, Otsu thresholding, contrast enhancement, grayscale
- **Dark background support** — auto-detects and inverts for white-on-dark text
- **100% offline** — uses Windows built-in OCR engine (Windows.Media.Ocr)
- **HiDPI / 4K ready** — Per-Monitor V2 DPI awareness
- **Multi-language** — supports all installed Windows OCR language packs
- **Lightweight** — ~5MB installer, no admin rights needed

## Install

Download the latest MSI from the [Releases](https://github.com/jrbuddysoftware/QuickSnipOCR/releases) page.

Requires Windows 10/11 with [.NET 8 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0).

## How It Works

1. Press `Ctrl+Shift+S` (or click **Capture Region**)
2. Drag to select the area containing text
3. Text is extracted and copied to your clipboard automatically

## Settings

Click the gear icon to configure:
- OCR language
- Auto-copy to clipboard
- License key (removes startup popup)
